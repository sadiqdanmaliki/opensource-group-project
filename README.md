# Student Result Management System

A web-based Student Result Management System built by **Group J** as part of the Opensource Software Development course at **Al-Qalam University Katsina**.

## 📌 Project Description

This system allows users to:
- Add students and their matric numbers
- Enter course scores
- Automatically calculate grades and GPA
- View and search student results
- Export/print result sheets

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
